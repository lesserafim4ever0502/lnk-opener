# lnkOpener Code Map

## Core Structure
- **package.json**: VSCode extension manifest declaring the `.lnk` custom editor provider and the `lnkOpener.showPreview` command and specific Markdown language hooks.
- **tsconfig.json**: TypeScript compiler settings.
- **src/extension.ts**: Extension entry point; registers both `LnkEditorProvider` and `MarkdownPreviewPanel`.
- **src/previewPanel.ts**: Manages the Webview lifecycle, advanced Markdown rendering (`markdown-it` with task lists), bulk PowerShell `.lnk` resolution for extreme performance, and Data URI encoding to bypass Webview local file restrictions for images.
- **out/**: Compiled JavaScript files (generated by `tsc`).
- ***.vsix**: The packaged extension available for installation.
